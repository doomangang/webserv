📁 File Upload Test Server

Upload File (using curl)

# Upload a text file
echo "Hello World!" > test.txt
curl -X POST -T test.txt http://127.0.0.1:8080/uploads/test.txt

# Upload binary file  
curl -X POST -F "file=@image.jpg" http://127.0.0.1:8080/uploads/image.jpg

# Upload with data
curl -X POST -d "This is test data" http://127.0.0.1:8080/uploads/data.txt
