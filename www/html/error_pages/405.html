405

허용되지 않는 메소드

요청한 리소스에 대해 사용된 HTTP 메소드가 허용되지 않습니다.
다른 메소드를 사용하여 다시 시도해 주세요.

HTTP 메소드 오류란?
• 서버가 해당 리소스에 대해 요청된 메소드를 지원하지 않음
• 예: POST만 허용하는 엔드포인트에 GET 요청
• 읽기 전용 리소스에 PUT/DELETE 요청
• CGI 스크립트가 특정 메소드만 처리하도록 설정됨
일반적으로 허용되는 HTTP 메소드:
• GET - 리소스 조회
• POST - 데이터 전송/생성
• PUT - 리소스 수정
• DELETE - 리소스 삭제
• HEAD - 헤더 정보만 조회
• OPTIONS - 지원 메소드 확인
🏠 홈으로 돌아가기 🔧 메소드 테스트

올바른 HTTP 메소드에 대한 정보가 필요하면 API 문서를 확인하세요.
