🐍 CGI 스크립트 테스트 센터

웹서버의 CGI (Common Gateway Interface) 기능을 테스트할 수 있는 다양한 스크립트들입니다.

ℹ️ CGI 테스트 정보

CGI 디렉토리: /cgi-bin/

지원 언어: Python, Shell Script

실행 권한: 모든 스크립트에 실행 권한이 부여되었습니다

🔧 사용 가능한 CGI 스크립트

🐍 Hello World (Python)
기본적인 Python CGI 스크립트입니다. 환경 변수와 쿼리 파라미터를 표시합니다.
기본 실행 파라미터 포함
📝 Form Handler (Python)
GET/POST 요청을 처리하는 스크립트입니다. 폼 데이터를 받아서 처리 결과를 보여줍니다.
GET 테스트 GET with 파라미터 폼 테스트 페이지
🔧 Environment Variables (Shell)
Shell 스크립트로 모든 환경 변수를 표시합니다. CGI 환경에서의 시스템 정보를 확인할 수 있습니다.
환경변수 보기
📁 Upload Handler (Python)
파일 업로드를 처리하는 스크립트입니다. multipart/form-data 형식의 파일 업로드를 테스트할 수 있습니다.
GET 요청 파일 업로드 페이지
